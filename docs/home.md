Get the tools you need to make native z/OS development _fun_.

Install zopen tools [Detailed Steps](/reference/install_zopen_tools.md):
 - [Download zopen-setup](https://zosopentools.link/setup-program) to your desktop
 - Upload _zopen-setup_ to z/OS with _sftp_
 - Log on to z/OS 
 - Mark _zopen-setup_ executable
 - Run _zopen-setup_

What does zopen tools do?
 - Installs tools you need that z/OS doesn't provide
 - zopen tools installs packages to their own directory and then symlinks their files into _/usr/local/zot_

Get started with:
 - [vim](/man/vim.html) : _zopen-install vim_
 - [bash](/man/bash.html): _zopen-install bash_
 - [git](/man/git.html) : _zopen-install git_
 - [cURL](/man/curl.html): _zopen-install curl_
